# Agent Rules and Preferences

- Prefer uv for Python dependency management
- Prefer Jujutsu (`jj`) for version control
- Prefer conventional commit format

## Style Guidelines

- When writing functions, avoid restricting the argument types (omit the argument types when possible). Declare argument types only when necessary for dispatch, correctness, or clarity.
- Prefer clear, concise, consistent, and understandable docstrings over verbose ones. Only provide an argument list when really necessary. Include additional explanations when they provide meaningful context (e.g., complex logic, non-obvious behavior, side effects).
